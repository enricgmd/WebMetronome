# Web Metronome

Metrónomo web simple, ejecutable localmente sin dependencias. También está preparado como PWA para instalar en móvil y funcionar offline después de abrirlo una vez.

## Uso

Abre `index.html` en un navegador o levanta un servidor local:

```bash
python3 -m http.server 8000
```

Después visita `http://localhost:8000`.

## PWA

Para usarlo fuera de casa, publica esta carpeta en un hosting estático con HTTPS, como GitHub Pages, Netlify o Vercel. En el móvil abre la URL publicada y usa la opción del navegador para añadirla a la pantalla de inicio.
